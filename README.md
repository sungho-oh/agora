# 🎯 숫자 맞추기 게임

1부터 100 사이의 랜덤한 숫자를 맞추는 간단하면서도 재미있는 웹 게임입니다!

## 🎮 게임 방법

1. 컴퓨터가 1부터 100 사이의 랜덤한 숫자를 선택합니다
2. 플레이어는 10번의 기회 안에 그 숫자를 맞춰야 합니다
3. 추측할 때마다 UP(더 큰 숫자) 또는 DOWN(더 작은 숫자) 힌트를 받습니다
4. 범위가 점점 좁혀지면서 정답을 찾아가세요!

## ✨ 주요 기능

- 🎲 랜덤 숫자 생성 (1-100)
- 🔢 10번의 추측 기회a
- 💡 UP/DOWN 힌트 제공
- 📊 실시간 범위 표시
- 📝 시도 기록 표시
- 🎨 반응형 디자인 (모바일 최적화)
- ⌨️ 키보드 지원 (Enter키로 추측)
